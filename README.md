# A Example Portfolio Homepage #

> To get start with you need the Visual Code extensions [Live SASS Compiler](https://marketplace.visualstudio.com/items?itemName=Syler.sass-indented) and [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).
>
> - Download the Zip and extract them into a disctionary.
> - Open the dictionary with [Visual Studio Code](https://code.visualstudio.com/).
> - Install the required extension and click in the bottom minibar *Go Live*.
